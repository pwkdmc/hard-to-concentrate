# Difficult to Focus
A responsive landing page exploring concentration challenges in the modern world. The site features a thematic design with dark/light mode switching capability.
## Features
- Theme Switching: Three color schemes (light, dark, auto) with persistent storage
- Responsive Design: Adapts to mobile, tablet, and desktop views
- Semantic HTML: Structured content with accessibility in mind
- CSS Architecture: Modular stylesheets with CSS variables
- Decorative Elements: Dynamic corner decorations and theme indicators
## Technologies
- HTML5
- CSS3 (Flexbox, Grid, Media Queries, Variables)
- JavaScript (Theme management with localStorage)
- Custom fonts (IBM Plex Mono)
## Structure
- Header: Theme controls, decorative REC indicator, main title with description
- Main Sections: 
  - Why concentration is difficult
  - Factors reducing attention
  - Practical advice
  - Image gallery
- Footer: Focus title
## Theme System
- theme-dark: Default dark theme with neon accents
- theme-light: Light theme with pastel accents
- theme-auto: Respects system preferences
## Browser Support
Modern browsers with CSS Grid and Custom Properties support
